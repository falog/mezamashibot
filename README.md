# mezamashibot
Alarm BOT for Discord

# どんなBOTか

朝８時ボイスに入って音楽ファイルを１回再生します。


音声ファイルは、各自用意してください。 サンプルとして公開しているものは、ご自由にお使いください。

# 使い方

node.jsをインストールしてください。  
その後以下のコマンドでpackageのインストールを行ってください。
> npm install

index.js内の以下の変数を設定してください。

YOUR_DISCORD_TOKEN  

TEXT_CHANNEL_ID 

VOICE_CHANNEL_ID 

MESSAGE


以下のコマンドで動かします。

> node index.js
